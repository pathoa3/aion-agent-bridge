# Pass654B Offline Capture Validation

Comprehensive offline validation of the current EuroAion test capture using only existing local PCAP and known-message log inputs. The pipeline merges oracle rows, dynamically inventories flows, reconstructs stream metadata, runs deterministic known-text tests, characterizes 10242 and the dynamic world flow, compares whisper/group controls, validates candidates, accounts for hypotheses, and writes a final decision.

Git artifacts contain scripts and high-level metadata only.
