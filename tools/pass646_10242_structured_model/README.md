# Pass646 10242 Structured Sidechannel Model

Consumes safe Pass645 10242 metadata artifacts and derives timing/length model candidates for the visible-chat sidechannel.

The script may inspect the local fresh pcap only to compute aggregate length and entropy metadata. It does not write packet payloads, hex dumps, packet hashes, decoded blobs, keys, binaries, DLLs, EXEs, tokens, or secrets.
