# Pass655 World Framing Marathon

Resumable offline-only world-flow framing and exact known-message validation pipeline. The master runner maintains `artifacts/pass655_work_queue.json`, resumes unfinished stages, records blockers with fallback completion, and writes only scripts plus high-level metadata to git.

Packet contents, decoded byte dumps, private account information, credentials, executable files, and local plaintext outputs are not committed.
