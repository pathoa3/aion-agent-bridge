# Pass608 Antigravity Next Bounded Hypotheses

We prepared a targeted set of next bounded hypotheses for Codex to test in the stream-cipher run:

## 1. Stream Cipher Hypothesis Categories

### Hypothesis 1: RC4 Stream Cipher
Aion login connections utilize RC4 packet encryption. If the EuroAion game-channel has reverted to or customized a stream mode, RC4 may be used.
- **Test parameters:**
  - Key: Derived from world seed `2D 66 BD 65` or lobby seed `19 1A 76 23`.
  - Sequential update: Simulate 15 intermediate C2S packet decryptions (discarding generated keystream bytes for intermediate packets).
  - Offsets: `4, 6, 8, 10`.

### Hypothesis 2: Blowfish Stream Modes (CFB, OFB, CTR)
If Blowfish is used, it must be in a mode where ciphertext size matches plaintext size without padding:
- **Blowfish CFB-8 or OFB-8:** Feed block state byte-by-byte.
- **Blowfish CTR:** Counter mode where keystream is generated by encrypting a counter.

### Hypothesis 3: Pure XORpass Stream Cipher
The packet may bypass Blowfish block encryption entirely, using only a custom LCG-updated XORpass.
- **Test parameters:** XORpass decryption only, with sequential key update over the 15 intermediate C2S packets.

## 2. Repeated Packet Differential Analysis
The two consecutive identical `KXSEQ_010_REPEAT` packets (Frames 4429 and 4435) provide a direct check:
- The XOR difference between their ciphertexts represents the XOR difference of the keystream blocks at those two states.
- If it is a stream cipher (RC4/CTR), the keystream blocks are generated independently of the plaintext, allowing direct differential checking.
